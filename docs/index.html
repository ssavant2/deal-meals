Deal Meals

Match current Swedish grocery offers with thousands of recipes and find meal ideas based on what is actually on sale this week.

View on GitHub Latest release

Built for Swedish groceries and recipes

Deal Meals fetches weekly offers from Swedish grocery chains, scrapes Swedish recipe sources, and matches ingredients against products with a Swedish-language matching engine.

Weekly deals

Scrapes offers from Willys, ICA, Hemkop, Mathem and Coop.

Recipe suggestions

Ranks recipes by savings, coverage and practical ingredient matches.

Self-hosted

Runs in Docker with a web UI, local cache, recipe images and settings.

What it looks like

The app focuses on the weekly workflow: fetch offers, gather recipes, review matches, and pick meals worth cooking.

Install with Docker

Download the Compose file and example environment, then start the app. The full documentation lives in the GitHub repository.

Deal Meals is a personal hobby project. Store and recipe scrapers may need maintenance when third-party websites change. 

mkdir deal-meals && cd deal-meals
wget -O docker-compose.yml https://github.com/ssavant2/deal-meals/releases/latest/download/docker-compose.yml
wget -O .env https://github.com/ssavant2/deal-meals/releases/latest/download/example.env
docker compose up -d
Deal Meals is open source on GitHub: github.com/ssavant2/deal-meals
